# 🎨 Teamplates Psico - Portfolios para Psicólogas

Sistema profesional y escalable para crear y gestionar templates de portfolios de psicólogas.

## 📁 Portfolios Disponibles

### 1. Portfolio 1 - Verde Minimalista
**Ubicación:** `templates/portfolio-1/`
- **Estilo:** Profesional y minimalista
- **Colores:** Verde salvia, tonos naturales
- **Ideal para:** Imagen corporativa seria y confiable

### 2. Portfolio 2 - Azul Moderno
**Ubicación:** `templates/portfolio-2/`
- **Estilo:** Moderno y contemporáneo
- **Colores:** Azul, indigo, slate
- **Ideal para:** Profesionales con enfoque en innovación

### 3. Portfolio 3 - Rosado Vibrante
**Ubicación:** `templates/portfolio-3/`
- **Estilo:** Colorido y expresivo
- **Colores:** Rosa, morado, tonos cálidos
- **Ideal para:** Comunicación cercana y acogedora

## ✨ Características de los Portfolios

✅ **Completamente Responsive**
✅ **Optimización de Imágenes HD** (sin cortes, calidad profesional)
✅ **Animaciones Suaves**
✅ **SEO Friendly**
✅ **Fácil Personalización**
✅ **Código Limpio**

## 🚀 Inicio Rápido

1. Elige el portfolio que más te guste
2. Navega a su carpeta: `templates/portfolio-X/`
3. Ejecuta `run.bat` o abre `index.html` directamente
4. ¡Listo! Tu portfolio está funcionando

## 🚀 Características

- ✅ Sistema modular y escalable
- ✅ Preparado para integración con base de datos
- ✅ Seguridad incorporada (autenticación, validación, encriptación)
- ✅ Múltiples templates profesionales
- ✅ API RESTful para futuras expansiones

## 🔧 Tecnología Recomendada

### Frontend
- React/Vue.js o Vanilla JS
- Tailwind CSS o Bootstrap para estilos
- Axios o Fetch API para peticiones

### Backend
- Node.js (Express) o Python (Flask/Django)
- JWT para autenticación
- Bcrypt para encriptación de contraseñas

### Base de Datos
- PostgreSQL o MySQL
- Mongoose (si usas MongoDB)

### Seguridad
- HTTPS obligatorio
- CORS configurado
- Rate limiting
- Validación de entrada
- SQL injection prevention
